WINDOWS AZURE MOBILE SERVICES

This quickstart app is already configured to work with your Mobile Service "ZUMOAPPNAME".

Supported browsers
==================
 - Internet Explorer 8+
 - Chrome
 - Firefox
 - Safari

If you use Chrome, Firefox, Safari or Internet Explorer 10+:
 - Simply open index.html in your browser (e.g., by double-clicking it).
   It will run from your local filesystem just the same as when hosted in a web server.

If you use Internet Explorer 9 or earlier:
 - For this quickstart to work, you'll need to host its files in a web server
   (e.g., IIS Express running on your workstation) and then browse to /index.html.
   Otherwise, the app will not be able to communicate with your Mobile Service.
